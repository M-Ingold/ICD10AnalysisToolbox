# Package index

## All functions

- [`DXCCSR`](https://m-ingold.github.io/ICD10AnalysisToolbox/reference/DXCCSR.md)
  : ICD-10-CM to Clinical Classifications Software Refined (CCSR)
  mapping
- [`create_tte_icd()`](https://m-ingold.github.io/ICD10AnalysisToolbox/reference/create_tte_icd.md)
  : Convert temporal ICD-10 codes into usable information for Cox
  regression
- [`fast_glm2()`](https://m-ingold.github.io/ICD10AnalysisToolbox/reference/fast_glm2.md)
  : Run many regression models quickly, for many response variables.
- [`phecodes`](https://m-ingold.github.io/ICD10AnalysisToolbox/reference/phecodes.md)
  : ICD-10-CM to PheCode mapping
- [`phewas_analysis_CCSR()`](https://m-ingold.github.io/ICD10AnalysisToolbox/reference/phewas_analysis_CCSR.md)
  : Determine the association of a variable of interest by running Cox
  regression on time to event information and logistic regression on
  prevalence at baseline. Generates PheWAS plots to assess the strenth
  of associations
- [`phewas_analysis_phecodes()`](https://m-ingold.github.io/ICD10AnalysisToolbox/reference/phewas_analysis_phecodes.md)
  : Determine the association of a variable of interest by running Cox
  regression on time to event information and logistic regression on
  prevalence at baseline. Generates PheWAS plots to assess the strenth
  of associations
- [`time_to_event_CCSR()`](https://m-ingold.github.io/ICD10AnalysisToolbox/reference/time_to_event_CCSR.md)
  : Map temporal ICD-10 codes into outcomes for Cox regression using the
  CCSR convention
- [`time_to_event_phecode()`](https://m-ingold.github.io/ICD10AnalysisToolbox/reference/time_to_event_phecode.md)
  : Map temporal ICD-10 codes into outcomes for Cox regression using the
  phecode convention
